Folder hiện tại là folder cha chứa 2 project con là:
- `echowave-backend`
    - viết bằng golang
- echowave-femodule
    - viết bằng typescript

Tại backend sẽ dùng command sau để run (mở terminal mới):
```
cd echowave-backend
docker compose up -d
go run ./playground
```

Tại frontend sẽ dùng command sau để run (mở terminal mới):
```sh
npm run dev
```

Hãy xem qua các file trong folder `echowave-backend/playground` và `echowave-femodule/src/pages/Example.tsx` để biết cách sử dụng các service trong backend và frontend.
Tính năng chính
- Backend sẽ host các endpoint liên quan đến websocket / long polling
    - backend cho phép lựa chọn dynamoDB hoặc postgresql làm database
- Frontend sẽ dùng src/context/provider.tsx và src/hooks/usePipewave.ts để tương tác với backend

Hãy xem cách sử dụng và giúp tôi viết 1 document Quick Start cho project này. Phải hướng dẫn thật dễ để tất cả mọi người (chưa biết golang/typescript) cũng có thể thực hiện được. Kết quả viết vào ./docpages/quickstart/*.md, đặt tên file hợp lý. Bạn có thể thêm nhiều ví dụ và lưu ý cho người dùng dễ hiểu.
